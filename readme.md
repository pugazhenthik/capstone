# Capstone E-Commerce Project

## Start server -- port 3000
```
npm start 
```

## APIs
### Authentication
    POST Register
    POST Login

### Users
    POST Add User
    GET Get User
    GET Get Users
    PUT Update User
    DEL Delete User

### Categories
    POST Add Category
    GET Get Category
    GET Get Categories
    PUT Update Category
    DEL Delete Category

### Products
    POST Add Product
    GET Get Product
    GET Get Products
    PUT Update Product
    DEL Delete Product

### Cart 
    POST Add to Cart
    PUT Update Cart
    PUT Checkout
    GET Get Cart Items

### Orders
    GET Orders
    GET Track Order
    PUT Update Order status

### Seller Reviews 
    GET Get Reviews
    POST Add Review
    GET Get Review
    PUT Update Review
    DEL Delete Review

### Product Reviews 
    GET Get Reviews
    POST Add Review
    GET Get Review
    PUT Update Review
    DEL Delete Review
